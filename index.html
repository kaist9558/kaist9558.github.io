What I Learned

Through this project, I learned that GitHub is more than a place to store code. It is a platform for organizing files, tracking changes, and collaborating with others. I also learned that GitHub Pages makes it possible to publish a website directly from a repository.

1. Understand the basic structure

I first learned the difference between GitHub, repositories, and GitHub Pages.

2. Build a minimal working version

I focused on making the site work first before trying to improve the design.

3. Improve through iteration

After the basic page worked, I refined the layout, text, and overall presentation.

Project Reflection

The most effective strategy was dividing the task into small steps. Instead of trying to understand everything at once, I learned one concept at a time: repository creation, file editing, publishing, and revision.

What worked well was using AI to explain unfamiliar terms and generate a starting structure. What did not work well was trying to make the page perfect from the beginning. I found that simplicity and gradual improvement were much more effective.
